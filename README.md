#  Pixel Todo

> Pixel art based todo app using vanilla JS, no framework, deployed on Netlify.
Elm Architecture pattern: model, update, view with localStorage

---

## Live Demo

Try it here:  
https://comfy-flan-6359bf.netlify.app

---

## Features

- Add, complete, delete tasks & filter by all-active-done
- 4 pastel themes (pink / yellow / blue / green) via color circles in the bottom bar
- Pixel art aesthetic · Press Start 2P font
- localStorage persistence — survives refresh
- Keyboard accessible 
- Mobile responsive
- Zero build step
